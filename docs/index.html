Reflection & Learning Journey

Throughout this course, I improved significantly in Android mobile development. Starting from basic UI creation in Lab 1, I gradually learned state management, navigation, local databases, cloud integration, and sensor interaction.


One of the biggest challenges was integrating multiple technologies together, especially Firebase Firestore and REST APIs. However, this project helped me understand real-world mobile application architecture and strengthened my debugging and problem-solving skills.


This project also deepened my understanding of SDG 3 by promoting healthy habits, hydration tracking, and wellness awareness through technology.
